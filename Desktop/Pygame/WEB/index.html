What makes it tick

Built like a notebook, plays like a shooter.

THE ZONE

It closes in. It always closes in.

Zone speed = 1.0 + level × 0.15 px/sec. It deals 9 HP/sec outside. The center drifts to a new random position each phase, so you can never just camp one corner. At Level 10 it moves at 2.5× base speed.

EVERY KID, NAMED

Pick a name. It shows up on every kill.

Live kill feed top-right. HP bar filled rect + outline. Armor vest overlay. ► YOU ◄ arrow above your head. Walk cycle, idle animations (reload, scratch, look, wipe face) — all drawn every frame via draw_figure().

GRENADES

2.5 seconds. Then everything around you dies.

Grenades travel at 260 px/sec with friction (0.97× per frame). 2.5 s fuse, then 22-particle explosion with gravity. Bots toss them at you if you're within 300 px. The Boss spams up to 3 at once in Berserk phase.

TOUCH CONTROLS

Joystick left. FIRE right. NADE above.

Virtual joystick radius 65 px (knob 26 px), FIRE button radius 38 px. Portrait → controls stack for split-screen. Landscape → P1 left half, P2 right half. Gamepad fully mapped: axes, shoot button, grenade button.

Input

Four ways to control the chaos.

Keyboard — Solo
  • WASD / Arrow keys → Move
  • SPACE → Shoot
  • G → Grenade
  • F11 → Fullscreen
  • F1 → Settings mid-match
  • ESC → Quit to menu
Keyboard — Split P1 / P2
  • P1: WASD + rebindable shoot
  • P2: Arrow keys + rebindable shoot
  • All keys rebindable in settings
  • Tab 2 = P1 binds, Tab 3 = P2 binds
Gamepad
  • Left stick → Move (deadzone 0.15)
  • Right stick → Aim
  • Configurable shoot / nade buttons
  • Axis mapping: 0–7 per axis
  • Sensitivity: 50–800 (default 300)
Touch (Mobile)
  • Left thumb → Virtual joystick
  • Right thumb → FIRE button
  • Above FIRE → NADE button
  • Top-right → Pause
  • Auto-aim toward nearest enemy
Enemy intelligence

Bots that actually fight back.

Every bot has its own AI loop — chasing you, running from the zone, dodging into buildings, and tossing grenades if you get too close.

AGGRO

Chase Range

Bots spot you at 500 px. Under 300 px they throw grenades. Over 500 px they wander randomly toward a new map point.

SCALE

Level Scaling

Lv1: 74 HP · 120 spd · 0.93s shoot CD. Lv10: 200 HP · 300 spd · 0.30s shoot CD. Speed boost ×1.4 when holding a BOOST loot.

ZONE

Zone Aware

If a bot strays outside the safe zone it sprints back at 1.3× speed. They won't camp outside — they'll die trying to reach you inside.

NADE

Grenade Throws

Grenade cooldown 10–25 s random. Bots throw at ±0.4 rad aim offset. With loot active: faster refire. Boss throws 1–3 grenades per burst.

Network play

LAN. Online. No excuses.

Two completely separate network stacks — UDP broadcast for the same room, WebSocket for the entire internet.

LAN — UDP

Host presses H → broadcasts BATTLEKIDS_HOST magic bytes every 1s on port 7655. Client presses J → listens up to 5s and auto-connects. Or type the IP manually. Port: 7654 TCP.

SAME ROOM · ZERO INTERNET

Online — WebSocket

Connects to Railway WSS server. Party rooms: public or password-locked, up to 10 players, 6-char room code. State messages broadcast HP, position, alive flag every frame.

WORLDWIDE · RAILWAY HOSTED

websockets auto-install

If the websockets Python package isn't found, the game installs it silently via pip on first launch. No setup from the player required. Falls back to offline modes instantly.

ZERO CONFIG · AUTO-SETUP
